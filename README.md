# WlanKey-Extractor

WLANKEY-EXTRACTOR (WKE)
A straightforward C++ utility for Windows designed to recover saved Wi-Fi passwords and network security details with zero hassle.

Why use this?
Instead of digging through complex Windows settings or running multiple CMD commands, this tool automates the process and filters the output so you only see the essential data.

Core Features

One-Step Extraction: Type the number next to any network to instantly see its password and security info.
Show All: One option to dump every saved Wi-Fi profile into a single, organized list.
Clean Output: Custom logic filters out redundant system information, showing each detail (Password, Cipher, Auth) only once.
No Compiler Needed: Provided as a ready-to-use .exe file. No programming knowledge or extra software required.
How to Use

Download the WlanKey-Extractor.exe file.
Right-click the file and select "Run as Administrator".
If a Windows protection message appears: Click "More info" -> "Run anyway" -> then select "Yes".
The tool will open instantly; choose a network number or select "Show All".
